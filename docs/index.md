# SongManager App

¡Bienvenido a la página oficial de soporte y documentación legal de **SongManager**!

SongManager es una aplicación móvil nativa construida en React Native diseñada para empoderar a cantantes, músicos y líderes de bandas. Permite la transposición matemática instantánea de acordes "Do Re Mi", exportación HD de partituras a WhatsApp y sincronización profunda auto-gestionada mediante Google Drive.

Actualmente, este repositorio aloja los documentos de transparencia requeridos para la verificación segura (OAuth) dentro de The Google Cloud Console.
